# SOP_05: Add-On Services Execution

**SOP Version:** 1.0  
**Date:** February 16, 2026  
**Purpose:** Procedures for executing add-on services consistently. Add-ons are optional services beyond standard common area cleaning.

---

## Add-On Service Types

### Add-On 1: Guest Suite Cleaning

**Description:** Deep clean of guest suite / short-term rental unit  
**Frequency:** Per request (not scheduled)  
**Pricing:** Per site Schedule B

**Scope includes:**
- Full unit cleaning (bedrooms, bathrooms, kitchen, living area)
- Linen change (if linens provided)
- Restock amenities (if provided)
- Trash removal
- Floor cleaning (vacuum/mop)
- Surface cleaning (dust, wipe)

**Scope excludes:**
- Personal belongings (do not move or clean personal items)
- Deep cleaning (carpet shampoo, oven clean) — separate service
- Repairs or maintenance

---

### Add-On 2: Elevator Move Clean

**Description:** Deep clean after elevator move (residents moving in/out)  
**Frequency:** Per request  
**Pricing:** Per site Schedule B

**Scope includes:**
- Elevator car deep clean (walls, floor, ceiling, fixtures)
- Remove debris and scuff marks
- Sanitize all surfaces
- Clean elevator lobby area (if affected)
- Photo evidence (before and after)

**Scope excludes:**
- Elevator mechanical work
- Moving assistance

---

### Add-On 3: Garbage Room Deep Clean

**Description:** Intensive cleaning of garbage room (beyond regular cleaning)  
**Frequency:** Monthly or per request  
**Pricing:** Per site Schedule B

**Scope includes:**
- Scrub floors (remove all residue and stains)
- Deep clean bins (inside and out; sanitize)
- Clean walls (remove splatters and residue)
- Clean ceiling and fixtures
- Remove cobwebs
- Deodorize thoroughly
- Photo evidence (before and after)

**Scope excludes:**
- Pest control (coordinate with site if needed)
- Bin replacement (admin coordinates)

---

### Add-On 4: Bio Cleanup

**Description:** Specialized cleanup of biohazards (blood, bodily fluids, etc.)  
**Frequency:** Per request (emergency)  
**Pricing:** Per site Schedule B (may include emergency surcharge)

**Scope includes:**
- Safe removal of biohazard materials
- Sanitization of affected area
- Proper disposal (per regulations)
- PPE required (gloves, mask, protective suit if needed)

**Scope excludes:**
- Needle disposal (coordinate with site/admin)
- Large-scale biohazards (coordinate with specialized crew)

**Note:** Only workers trained in biohazard cleanup may perform this add-on.

---

## Add-On Execution Procedure

### Pre-Start

**Step 1.1:** Verify add-on approved:
- Check work order status: APPROVED
- Verify pricing and scope per Schedule B
- Confirm worker assigned (if applicable)

**Step 1.2:** Review add-on checklist:
- Each add-on has specific checklist
- Review items before starting

**Step 1.3:** Gather supplies:
- Standard cleaning supplies
- Add-on-specific supplies (if needed)
- PPE (if bio cleanup)

---

### Execution

**Step 2.1:** Take before photos:
- Document condition before cleaning
- Show areas that need attention
- Required for add-ons (before/after proof)

**Step 2.2:** Execute add-on per checklist:
- Complete each checklist item
- Follow add-on-specific procedures
- Take photos during process (if needed)

**Step 2.3:** Take after photos:
- Document completion
- Show cleaned areas
- Required for add-ons (before/after proof)

**Step 2.4:** Complete add-on checklist:
- Mark all items PASS / FAIL / NA
- Add notes if needed
- Submit completion

---

### Post-Execution

**Step 3.1:** Submit add-on completion:
- Checklist completed
- Before/after photos uploaded
- Notes added

**Step 3.2:** Work order status: COMPLETED  
**Step 3.3:** Admin reviews and marks INVOICED  
**Step 3.4:** Client invoiced per Schedule B pricing

## Pricing and Approval

### Pricing

**Per site Schedule B:**
- Each add-on has fixed pricing
- Emergency surcharge may apply (bio cleanup, after-hours)
- Pricing visible to admin only (workers do not see pricing)

### Approval Process

**Step 4.1:** Client requests add-on (via property management)  
**Step 4.2:** Admin creates work order:
- Type: Add-on service
- Description: Specific add-on requested
- Pricing: Per Schedule B
- Status: REQUESTED

**Step 4.3:** Admin approves work order:
- Status: APPROVED
- Assigns to worker (if applicable)
- Worker notified

**Step 4.4:** Worker executes add-on per this SOP  
**Step 4.5:** Admin reviews completion and marks INVOICED  
**Step 4.6:** Client invoiced per Schedule B

---

## Decision Tree: Add-On Issues

### Issue: Add-On Scope Unclear
→ **Review Schedule B** → **Contact admin** → **Clarify scope** → **Proceed**

### Issue: Supplies Missing for Add-On
→ **Document in notes** → **Contact admin** → **Admin coordinates supplies** → **Reschedule if needed**

### Issue: Biohazard Beyond Training
→ **STOP WORK** → **Document incident** → **Escalate to admin** → **Admin coordinates specialized crew**

### Issue: Client Requests Additional Work
→ **Do not perform** → **Document request** → **Escalate to admin** → **Admin creates new work order if approved**
