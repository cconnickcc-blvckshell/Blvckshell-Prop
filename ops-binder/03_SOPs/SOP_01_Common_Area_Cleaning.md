# SOP_01: Common Area Cleaning

**SOP Version:** 1.0  
**Date:** February 16, 2026  
**Purpose:** Step-by-step order of operations for common area cleaning. Ensures consistency, safety, and quality.

---

## Pre-Start Checklist

Before beginning cleaning:

- [ ] Review job assignment: site, scheduled time, checklist template
- [ ] Verify access credentials (key, FOB, code) are available
- [ ] Check required supplies and equipment (per site agreement)
- [ ] Review site-specific access instructions and service window
- [ ] Ensure PPE is available (gloves, mask if needed)
- [ ] Mobile device charged for photo evidence

---

## Order of Operations

### 1. Arrival and Access

**Step 1.1:** Arrive at scheduled time (within service window)  
**Step 1.2:** Use access credentials per site instructions  
**Step 1.3:** If access denied:
- Contact admin immediately
- Do not force entry
- Document in incident report

**Step 1.4:** Enter site and verify you're in correct location

---

### 2. Safety and Signage

**Step 2.1:** Place "Cleaning in Progress" signage if required by site  
**Step 2.2:** Check for safety hazards:
- Wet floors
- Broken fixtures
- Exposed wires
- Other hazards

**Step 2.3:** If safety hazard found:
- **STOP WORK**
- Document via incident report (type: SAFETY)
- Contact admin immediately
- Do not proceed until hazard resolved

---

### 3. Gather Tools and Supplies

**Step 3.1:** Verify supplies provided per site agreement:
- Cleaning chemicals (appropriate for surface types)
- Mops, buckets, microfiber cloths
- Vacuum (if carpeted areas)
- Trash bags
- PPE (gloves, mask)

**Step 3.2:** If supplies missing or insufficient:
- Document in notes
- Contact admin
- Proceed with available supplies if safe

---

### 4. Start-to-Finish Route

Follow this route to minimize backtracking and ensure all areas covered:

**Route Order:**
1. **Lobby** (CL_01)
2. **Hallways** (CL_02)
3. **Stairwells** (CL_03)
4. **Elevators** (CL_04)
5. **Glass doors** (CL_06 — spot clean)
6. **Garbage rooms** (CL_05)
7. **Washrooms** (CL_07)

**Note:** Adjust route per site layout, but maintain logical flow (start at entrance, work inward).

---

### 5. Area Cleaning Procedure (Per Checklist)

For each area, follow the checklist template assigned to that site:

**Step 5.1:** Open checklist template on mobile device  
**Step 5.2:** Complete each checklist item in order:
- Perform task
- Mark PASS / FAIL / NA
- Add note if needed (e.g., "bulb needs replacement")

**Step 5.3:** Take required photos:
- Minimum photo count per area (see checklist)
- Mandatory photo items (e.g., floor, bins)
- Ensure photos are clear and show completion

**Step 5.4:** Verify "definition of done" met:
- No visible debris, streaks, or residue
- All required items marked PASS
- Minimum photo count met

**Step 5.5:** If any item marked FAIL:
- Document reason in notes
- Attempt correction if possible
- If cannot correct, mark FAIL and note reason

---

### 6. Cleaning Chemicals and Techniques

**Floor Cleaning:**
- **Tile/Concrete:** Sweep → mop with appropriate cleaner → dry mop if needed
- **Hardwood:** Damp mop only (no excess water); use hardwood-specific cleaner
- **Carpet:** Vacuum thoroughly; spot clean stains with carpet cleaner

**Glass Cleaning:**
- Use glass cleaner and lint-free cloth or squeegee
- Clean both sides if accessible
- Check for streaks from multiple angles

**Sanitization:**
- High-touch surfaces: door handles, light switches, elevator buttons, handrails
- Use sanitizing cleaner or disinfectant wipes
- Allow contact time per product instructions

**Waste Disposal:**
- Empty bins completely
- Replace liners if provided
- Clean bin inside and out
- Sanitize bin lids and handles

---

### 7. Required Photos

**Photo Requirements (per area):**
- **Lobby:** Floor, glass doors, desk, bins (minimum 4)
- **Hallway:** Floor, baseboards, bins, overall view (minimum 4)
- **Stairwell:** Steps, handrails, baseboards, overall view (minimum 4)
- **Elevator:** Floor, control panel, walls, mirror (minimum 4)
- **Garbage Room:** Floor (mandatory), bins (mandatory), walls, overall view (minimum 4)
- **Glass Doors:** Glass (mandatory), frames, threshold (minimum 3)
- **Washroom:** Floor (mandatory), toilets (mandatory), sinks, mirrors (minimum 4)

**Photo Quality:**
- Clear, well-lit
- Show completion (e.g., clean floor, empty bins)
- Timestamped automatically by device
- No resident faces or unit numbers visible (if avoidable)

---

### 8. Completion and Submission

**Step 8.1:** Review all checklist items:
- All required items answered (PASS / FAIL / NA)
- All notes added where needed
- No TBD or incomplete items

**Step 8.2:** Verify photo evidence:
- Minimum photo count met per area
- Mandatory photo items captured
- Photos clear and show completion

**Step 8.3:** If checklist incomplete or photos insufficient:
- Complete missing items
- Take additional photos
- Do not submit until complete

**Step 8.4:** Submit completion:
- All checklist items answered
- Minimum photo count met
- Within photo cap (20 per job)
- Notes added where needed

**Step 8.5:** Job status transitions: SCHEDULED → COMPLETED_PENDING_APPROVAL

---

## Definition of Done (Per Area)

Each area is considered clean when:

- All required checklist items marked PASS
- No visible debris, streaks, smudges, or residue
- Bins emptied and clean (if applicable)
- High-touch surfaces sanitized
- No unpleasant odors
- Minimum photo count met
- Mandatory photo items captured

---

## Decision Tree: What If...

### Access Denied
→ Contact admin → Document incident → Do not proceed

### Safety Hazard Found
→ STOP WORK → Document incident (type: SAFETY) → Contact admin → Wait for resolution

### Supplies Missing
→ Document in notes → Contact admin → Proceed if safe with available supplies

### Checklist Item Cannot Be Completed
→ Mark FAIL → Add note explaining reason → Contact admin if critical

### Photo Upload Fails
→ Retry upload → If still fails, save draft → Contact admin → Resume later

### Resident Interruption
→ Be professional and courteous → Explain you're cleaning → Continue when safe

### Maintenance Issue Found
→ Document in notes → Report via incident report if safety-related → Continue cleaning

## Post-Cleaning

**Step 9.2:** Secure site (lock doors, return keys if applicable)  
**Step 9.3:** Submit completion via portal  
**Step 9.4:** Verify submission successful (job status: COMPLETED_PENDING_APPROVAL)
